# Base de datos

- `schema.sql`: estructura base para crear la base de datos.
- `productos_2026-05-20.sql`: carga de productos.
- `seed_*.sql`: datos demo o iniciales.
- `alter_*.sql`: cambios incrementales de tablas existentes.
- `remove_*.sql`: scripts de limpieza puntual.

No borrar scripts historicos sin confirmar que ya no se necesitan para reinstalar o migrar otra maquina.
